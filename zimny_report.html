Správa o postupe realizácie projektu: Game Recommender System

Autor: Ruslan Yehorov
Obdobie: Semester 1

1. Úvod a ciele semestra

Hlavným cieľom tohto semestra bolo položenie pevných technologických základov pre vývoj inteligentného systému na odporúčanie videohier. Kľúčovou úlohou bol návrh udržateľnej a škálovateľnej architektúry, ktorá umožní v ďalších fázach efektívne trénovanie modelov strojového učenia (LLM).

Stanovené ciele:

  • Inicializácia vývojového prostredia a nastavenie verziovania kódu.
  • Výber a technickú analýzu zdroja dát (RAWG API).
  • Návrh a implementáciu databázovej schémy.
  • Vývoj automatizovaného ETL procesu (Extract, Transform, Load).

2. Hĺbková analýza projektu

Pred začiatkom implementácie bola vykonaná dôkladná a ґruntovná analýza požiadaviek projektu s cieľom zabezpečiť vysokú kvalitu realizácie. Analyzovali sme štruktúru dát potrebnú pre budúce trénovanie neurónových sietí a zvolili stratégiu "Detail Fetching" pre maximalizáciu informačnej hodnoty každého záznamu pri zachovaní efektivity API limitov.

3. Dosiahnuté výsledky a technická realizácia

Všetky naplánované ciele boli úspešne splnené. Nižšie uvádzame detailný prehľad implementovaných komponentov:

Vývojové prostredie a Version Control

  • Konfigurácia izolovaného prostredia Python (venv) pre správu závislostí.
  • Inicializácia Git repozitára a implementácia .gitignore pre bezpečnosť projektu.

Databázová architektúra

  • Návrh relačnej schémy pre SQLite databázu optimalizovanú pre čítanie dát.
  • Tabuľka games obsahuje 19 atribútov vrátane JSON-serializovaných metadát.
  • Implementácia DDL skriptov s ochranou proti duplicite dát.

ETL Pipeline (Data Engineering)

  • Vývoj Python skriptov pre komunikáciu s RAWG API.
  • Implementácia pokročilého čistenia textu (Regex) pre atribút description (odstránenie HTML tagov).
  • Optimalizácia zápisu do databázy pomocou transakcií a metódy executemany.
  • Spracovanie chybových stavov (Error Handling) pre zabezpečenie stability procesu.
